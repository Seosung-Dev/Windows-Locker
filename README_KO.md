## 환경 설정

1. `.env.example` 파일을 `.env`로 복사하세요.

Windows:
```
copy .env.example .env
```

Linux / Mac:
```
cp .env.example .env
```

2. `.env` 파일을 열고 값을 설정하세요.

```
ID=your_id
PASSWORD=your_password
```

3. 실행하세요
