# How to use
## Installation

```
pip install python-dotenv screeninfo
```

---

## Environment Setup

1. Copy `.env.example` to `.env`

Windows:
```
copy .env.example .env
```

Linux / Mac:
```
cp .env.example .env
```

2. Edit the `.env` file and set your values:

```
ID=your_id
PASSWORD=your_password
```

3. The `.env` file is excluded from Git for security reasons.
